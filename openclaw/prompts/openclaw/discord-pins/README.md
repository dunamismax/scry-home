# Discord Home-Channel Pinned Notes

Canonical source files for the live pinned notes in the Discord agent home channels.

Use these when posting or refreshing the pinned note for a specific channel:

- `scry.md` → `#scry`
- `codex.md` → `#codex`
- `research.md` → `#research`
- `scribe.md` → `#scribe`
- `operator.md` → `#operator`
- `sentinel.md` → `#sentinel`
- `luma.md` → `#luma`

Rules:

- Each file should mention only the prompts that actually belong in that channel.
- Keep the copy short enough to work as a real pinned note, not a mini handbook.
- If prompt selection or home-channel routing changes, update the affected file(s), `../README.md`, and any global overview docs that still mention the old flow.
- `../DISCORD_PINNED_QUICKSTART.md` is now a global overview, not the source for live per-channel pins.
