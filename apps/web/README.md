# @scry-home/web

TanStack Start frontend for the optional `scry-home` workspace UI.

## Scripts

```bash
bun run --filter @scry-home/web dev
bun run --filter @scry-home/web typecheck
bun run --filter @scry-home/web build
```

## Notes

- Routes are file-based under `src/routes`.
- `src/routeTree.gen.ts` is generated by TanStack Router.
- Server-only logic belongs under `src/server` or API route handlers.
- Treat this app as a helper surface for the Scry repo, not the repo's primary product.
