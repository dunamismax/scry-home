# @scry-home/web

TanStack Start frontend for the `scry-home` control plane.

## Scripts

```bash
pnpm --filter @scry-home/web dev
pnpm --filter @scry-home/web typecheck
pnpm --filter @scry-home/web build
```

## Notes

- Routes are file-based under `src/routes`.
- `src/routeTree.gen.ts` is generated by TanStack Router.
- Server-only logic belongs under `src/server` or API route handlers.
